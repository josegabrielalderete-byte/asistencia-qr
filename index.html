📋 Instrucciones: Conectar Google Sheets

  1. Crea una hoja de cálculo en Google Sheets.
  2. Agrega los encabezados: Nombre | Matrícula | Fecha | Hora | Estado
  3. Ve a Extensiones → Apps Script y pega el siguiente código:
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data  = JSON.parse(e.postData.contents);
  sheet.appendRow([data.nombre, data.matricula, data.fecha, data.hora, data.estado]);
  return ContentService
    .createTextOutput(JSON.stringify({ok: true}))
    .setMimeType(ContentService.MimeType.JSON);
}
